# Features (قابلیت‌ها)

مستندات قابلیت‌های خاص پروژه همسو.

## فایل‌ها

- **01-ai-report.md** - راهنمای قابلیت گزارش هوشمند AI
- **02-pdf-export.md** - راهنمای خروجی PDF فارسی

## برای کی این بخش است؟

- توسعه‌دهندگان که روی فیچرهای خاص کار می‌کنند
- هر کسی که می‌خواهد قابلیت خاصی را پیاده‌سازی یا تغییر دهد
- QA engineers برای تست فیچرها

## قابلیت‌های موجود

### AI Report
- تولید گزارش هوشمند از پیشرفت کاربر
- تحلیل آماری و sentiment analysis
- پیشنهادات عملی

### PDF Export
- خروجی PDF از گزارش‌ها
- پشتیبانی کامل از زبان فارسی
- استایل‌بندی حرفه‌ای
