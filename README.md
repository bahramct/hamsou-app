**همسو** یک پلتفرم شخصی و اجتماعی برای همسویی و رشد فردی است که به شما کمک می‌کند تا:

- 🎯 تعهدات روزانه خود را تعیین و پیگیری کنید
- 📝 در پایان هر روز بازتفل (Reflection) عمیق داشته باشید
- 📅 برنامه‌های هفتگی و ماهانه هدفمند تدوین کنید
- 🤖 از هوش مصنوعی برای تحلیل پیشرفت و دریافت بینش‌های ارزشمند استفاده کنید
- 🔔 نوتیفیکیشن‌های هوشمند و به‌موقع دریافت کنید
- 📊 پیشرفت خود را با نمودارها و گزارش‌های تحلیلی بررسی کنید
- 👥 در جامعه‌ای از افراد با اهداف مشابه عضو شوید و از هم انگیزه بگیرید
- 🏆 در چالش‌های گروهی شرکت کنید و در لیدربورد رقابت سالم کنید

| 👔 مقایسه و حسادت | 🤝 انگیزه و همراهی |
| ⭐ لایک و کامنت | 👏 تشویق و حمایت |

## ✨ ویژگی‌های کلیدی

### 🎯 تعهدات روزانه (Daily Commitments)
<div dir="rtl">

- تعریف و پیگیری تعهدات روزانه با دسته‌بندی و اولویت‌بندی
- اتصال تعهدات به برنامه‌های هفتگی و ماهانه
- نمایش تاریخچه کامل از تعهدات انجام شده و انجام نشده
- نوتیفیکیشن‌های هوشمند یادآوری تعهدات
</div>

### 📝 بازتاب روزانه (Daily Reflections)
<div dir="rtl">

- فرم بازتاب روزانه با سوالات هدایت‌کننده
- ثبت موفقیت‌ها، چالش‌ها، یادگیری‌ها و شکرگزاری
- تحلیل خلق‌وخوی روزانه با نمودارهای بصری
- تولید گزارش هفتگی خودکار توسط هوش مصنوعی
- شناسایی الگوهای رفتاری و پیشنهادهای بهبود
</div>

### 📊 تحلیل و گزارش‌دهی پیشرفته (Advanced Analytics)
<div dir="rtl">

- داشبورد تحلیلی کامل با نمودارهای تعاملی
- نرخ تکمیل تعهدات و روندهای زمانی
- نقشه فعالیت روزانه (Heatmap) به سبک GitHub
- ابر کلمات (Word Cloud) برای تحلیل تعهدات و بازتاب‌ها
- خروجی Excel از گزارش‌ها
- فیلتر زمانی (هفته، ماه، همه زمان)
</div>

### 🤖 بینش‌های هوش مصنوعی (AI-Powered Insights)
<div dir="rtl">

- تولید خودکار گزارش هفتگی با AI
- شناسایی الگوهای رفتاری و عادات
- پیشنهادهای شخصی‌سازی شده برای بهبود
- تحلیل احساسی بازتاب‌ها (در آینده)
- دستیار هوشمند برای پشتیبانی (در آینده)
</div>

### 👥 قابلیت‌های اجتماعی (Social Features)
<div dir="rtl">

- **پروفایل کاربری**: نمایش پیشرفت شخصی با آمار و دستاوردها
- **اشتراک‌گذاری دستاوردها**: تولید تصویر از دستاوردها برای اشتراک‌گذاری در سوشال مدیا
- **فید جامعه**: دنبال کردن پیشرفت دیگران، لایک و کامنت
- **لیست کاربران**: جستجو، فالو و آنفالو کاربران
- **چالش‌های گروهی**: شرکت در چالش‌های تیمی برای انگیزه بیشتر
- **لیدربورد**: رقابت سالم و سالم (هفتگی، ماهانه، همیشه)
- **سیستم امتیازدهی**: هر پست ۱۰ امتیاز، هر لایک دریافتی ۲ امتیاز، هر کامنت دریافتی ۳ امتیاز، هر فالوور ۱ امتیاز

## 🚀 شروع سریع

### پیش‌نیازها

- Node.js 18+
- Bun (توصیه می‌شود) یا npm/pnpm/yarn
- Git
- مرورگر مدرن (Chrome, Firefox, Safari, Edge)

### نصب و راه‌اندازی

```bash
# کلون کردن ریپوزیتوری
git clone https://github.com/bahramct/hamsou-app.git
cd hamsou-app

# نصب وابستگی‌ها
bun install

# تنظیم متغیرهای محیطی
cp .env.example .env
# .env را با تنظیمات خود ویرایش کنید

# راه‌اندازی دیتابیس
bun run db:push

# اجرای سرور توسعه
bun run dev
```

مرورگر را باز کنید: [http://localhost:3000](http://localhost:3000)

### متغیرهای محیطی

```env
# دیتابیس
DATABASE_URL="file:./db/hamsou.db"

# برنامه
NEXT_PUBLIC_APP_URL="http://localhost:3000"

# JWT (یک رشته تصادفی تولید کنید)
JWT_SECRET="your-secret-key-here-change-this-in-production"
```
